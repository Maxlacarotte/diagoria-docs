Installer DIAGORIA comme un vrai logiciel 💿

Un installeur Windows propre : DIAGORIA s'ajoute au menu Démarrer, avec un désinstalleur — fini le .exe qui traîne sur le bureau.

⚡ En 30 secondes

Télécharge l'installeur, clique « Installer », et DIAGORIA apparaît dans ton menu Démarrer (+ désinstalleur dans « Ajout/Suppression de programmes »). Au 1ᵉʳ lancement, il s'ouvre tout seul dans ton navigateur.

⬇️ Télécharger l'installeur (Windows) Version portable (.exe)

🧭 Les 3 étapes

  1. Télécharge l'installeur

    DIAGORIA-Setup-2.1.0.exe (~13,5 Mo) depuis le bouton ci-dessus ou la vitrine diagoria.com.

  2. Lance-le et clique « Installer »

    Windows peut afficher un avertissement bleu (« éditeur inconnu ») car le logiciel n'est pas encore signé → clique « Informations complémentaires » puis « Exécuter quand même ». Comment supprimer cet avertissement →

  3. C'est prêt

    DIAGORIA est dans ton menu Démarrer. Au lancement, le moteur démarre et ton navigateur s'ouvre tout seul sur l'application.

📍 Où ça s'installe

Dans %LOCALAPPDATA%\Programs\DIAGORIA (installation par utilisateur, sans mot de passe administrateur). On obtient quand même tout ce qui fait un « vrai logiciel » : entrée menu Démarrer, raccourci bureau (optionnel), désinstalleur propre dans « Ajout/Suppression de programmes ».

⚙️ Pourquoi pas directement dans « Program Files » ?

Parce que DIAGORIA écrit ses fichiers (compte, dossiers patients) à côté de l'exécutable, et que « Program Files » est en lecture seule pour un utilisateur normal — l'app ne pourrait pas sauvegarder. Pour une vraie install Program Files (tous utilisateurs, avec admin), il faut d'abord déplacer les données vers %APPDATA%\DIAGORIA — une petite évolution prévue côté logiciel.

🍎🐧 Mac & Linux

Une chaîne de compilation automatique (GitHub Actions) a été mise en place pour produire les versions macOS et Linux en plus de Windows. Dès qu'elles sont validées, elles seront proposées au téléchargement sur la vitrine. En attendant : « bientôt disponibles ».

🗑️ Désinstaller

Paramètres Windows → Applications → DIAGORIA → Désinstaller (ou via le menu Démarrer). Tes dossiers patients ne sont pas supprimés automatiquement (donnée médicale) — tu les enlèves toi-même si tu le souhaites.
