DISTRIBUTION WINDOWS

Supprimer l'avertissement SmartScreen

Pourquoi Windows bloque l'installation de DIAGORIA — et comment faire confiance à l'éditeur.

⚡ En 30 secondes

Windows affiche « Éditeur inconnu » parce que le .exe n'est pas signé avec un certificat Authenticode reconnu. La solution la plus simple et la moins chère : certificat OV (Organisation) chez Certum — environ 30 €/an en renouvellement (69 € la 1ʳᵉ année avec le token matériel inclus). L'alerte disparaît progressivement à mesure que DIAGORIA accumule des téléchargements propres. Si tu veux zéro alerte dès le 1ᵉʳ téléchargement : certificat EV (~200–300 €/an) — mais depuis 2024 Microsoft a réduit cet avantage. Délai pour obtenir un certificat OV : 1 à 5 jours ouvrés.

1 — Pourquoi l'alerte apparaît

Quand tu télécharges un .exe depuis Internet, Windows lit un marqueur invisible attaché au fichier (Zone.Identifier). Il envoie ensuite son empreinte au service SmartScreen de Microsoft, qui répond : « Ce programme est-il connu ? Propre ? »

Si le programme est inconnu (nouveau, peu téléchargé, ou sans signature), SmartScreen affiche le fameux écran rouge/gris. Ce n'est pas un virus — c'est un manque de réputation.

La réputation s'acquiert de deux façons : (1) beaucoup de téléchargements sans plainte, ou (2) une signature de code Authenticode — un tampon cryptographique qui prouve que Maximilien Vergnaud a bien produit ce fichier et qu'il n'a pas été modifié depuis.

Authenticode = technologie Microsoft (intégrée depuis Windows XP) qui permet d'apposer une signature numérique sur un exécutable. Un certificat délivré par une Autorité de Certification (AC) reconnue est nécessaire.

Certum inclut ce token dans son offre "set" dès la 1ʳᵉ commande.

4 — Faisable en entreprise individuelle (EI) ?

Oui. Pour un certificat OV, l'autorité de certification vérifie que tu existes en tant qu'entité légale. En France, cela signifie :

  • Ton numéro SIREN (visible sur le Kbis ou avis de situation INSEE)
  • Parfois une preuve de domaine (ex : domaine diagoria.fr que tu possèdes)
  • Une pièce d'identité / justificatif professionnel selon l'AC

Pour un certificat EV, certaines AC demandent un numéro DUNS (identifiant D&B international). Il est gratuit à obtenir sur le site de Dun & Bradstreet mais peut prendre 1–4 semaines. Certum EV semble ne pas l'exiger systématiquement pour les EI françaises — vérifier directement avec eux.

DUNS = Data Universal Numbering System — identifiant à 9 chiffres attribué par Dun & Bradstreet. Sert à prouver l'existence d'une entité commerciale au niveau international. Gratuit, demande en ligne sur dnb.com.

MSIX = format de package Windows moderne (comme un .zip intelligent). Il remplace les vieux installeurs .msi. Avantage : installation propre, désinstallation complète, pas de "résidus" registre. Inconvénient : nécessite de refactorer le packaging de l'app.

6 — Recommandation & parcours étape par étape

Reco pour DIAGORIA (phase actuelle) — Certificat OV Certum "Open Source" si DIAGORIA peut être qualifié d'open source, sinon OV Standard Certum. Budget : 69 € la 1ʳᵉ année (token + certificat), puis 29–80 €/an selon l'offre choisie. L'EV ne vaut pas 3× le prix tant que la base d'utilisateurs est petite — la réputation SmartScreen se bâtit de toute façon progressivement depuis 2024.

  1. OBLIGATOIRE — Commande le "set" Certum OV

    Va sur shop.certum.eu. Choisis l'offre "Open Source Code Signing" (si applicable) ou "Standard". Commande le set complet (= certificat + carte à puce + lecteur USB). Coût : ~69 €. Délai livraison token : 3–7 jours.

  2. OBLIGATOIRE — Fournis les justificatifs de ton EI

    Prépare : avis de situation SIREN (disponible sur annuaire-entreprises.data.gouv.fr), pièce d'identité, éventuellement preuve de domaine diagoria.fr. L'AC te contacte par mail ou téléphone. Délai validation : 1–5 jours ouvrés.

  3. OBLIGATOIRE — Signe le .exe avec signtool.exe

    Une fois le certificat chargé sur la carte à puce, utilise signtool.exe sign /tr http://timestamp.certum.pl /td sha256 /fd sha256 /a diagoria.exe depuis le SDK Windows. Le timestamp est obligatoire — sans lui, la signature expire en même temps que le certificat. Intègre ça dans le script de build.

  4. CONSEILLÉ — Rebuilde en MSIX pour Windows 11

    Si tu veux une installation "propre" et une meilleure intégration Windows, repackage en MSIX avec MSIX Packaging Tool (gratuit). Signe avec le même certificat Certum.

  5. FACULTATIF — Publication sur le Microsoft Store

    Idéal à terme pour zéro alerte et découvrabilité accrue. Compte développeur : 19 $ une fois. Processus : soumettre le MSIX, attendre validation Microsoft (quelques jours). Microsoft re-signe le package — plus besoin de certificat propre pour la version Store.

  6. FACULTATIF — Inscription dans winget

    Ouvre une Pull Request sur le dépôt winget-pkgs avec le manifest de DIAGORIA. Permet aux utilisateurs tech d'installer via winget install diagoria.
